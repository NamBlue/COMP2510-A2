COMP 2510 - Assignment 2 - Character Translator

Mimics the tr command in Unix systems.

Runs in:

./a2.exe [argument1] [argument2]

argument1 - character set to translate
argument2 - destination character set to translate to

e.g a2 "a-c\n" "x-z1" 

arguments with escape sequences should be wrapped in quotes
use '-' to specify a range of characters
